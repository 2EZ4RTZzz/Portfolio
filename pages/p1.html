Childhood memory

Happy Nanji
2D, side-scrolling role-playing game

This a side-scrolling 2D cartoon-style game in which the protagonist progresses through fights, challenges of varying difficulty levels, and eventually kills bosses to achieve success in the game. The idea for the game was inspired by a game called"MapleStory" I loved to play as a kid, and it was my first game as a freshman at SFU.

Check out my projects!

Concept:

This game's concept is about a Horizontal 2D cartoon style upgrade class game.

How To play:

For the movement, you can press the arrow key to make the character move, press the Space bar can control the character to attack (shot the ammo) to the enemy, and the damage will increase based on the character's level. There will be two bars at the top left of the screen, a red bar for HP and a green bar for EXP. The right bar includes the current exp state, current attack damage, and current level.

About the spells

Skill1 . press Q to increase the 20% extra attack damage in 10 secs Skill2. Press F to recovery 2hp , each level can only cast 3 times.

Problem:

1. There are many bugs in the game, some of which cause the character to get stuck, so I finally solved the bugs through a lot of study and testing.

2. Basic knowledge about how to code collision and make the platform to make sure the player can stay on the correct location on the platform. Game Balance issue, Player level up problem , player attack correct position..etc

Design process:

Week1. Start making the game uml and draw all the game images. Week2. Working on the each class code and test it Week3. Working on the challenge part(for example: collision , amno check , speed , platform issue) Week4. Add background songs, attack songs , add attack effect , level up effect..etc
